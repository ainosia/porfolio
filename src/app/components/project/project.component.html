Red Social MEAN

MongoDB, Express.js, Angular, Node.js

Como Twitter

Este proyecto pertenece a una serie de aplicaciones realizadas combinando Angular con otras tecnologías (en este caso MogoDB, Express.js y Node.js) y está llevada a cabo tanto a nivel de frontend como de backend, empleando únicamente JavaScript.

Es una red social similar a Twitter, que incluye control de acceso y que permite registrar usuarios, añadir publicaciones y seguidores, seguir a otros usuarios, etc.

https://github.com/Social-Mean
